Backups are useless

OpenSiteBackup is a professional, restore-first backup pipeline for website operators. It automates backup, integrity verification, storage upload, and staged restore drills with safety gates.

1) Backup + Verify

Captures files + database, validates archive integrity, SQL sanity, and checksums before upload.

2) Upload Backends

Use local by default. Optional cloud via rclone or gog. Optional backend failures don’t block required local baseline.

3) Restore Confidence

Runs staged restore checks before swap and keeps rollback path. Emits RESTORE_SUMMARY markers for evidence.

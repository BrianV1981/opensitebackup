# Positioning & GTM

## Core Positioning
"Open-source backup engine you can inspect, with restore workflows you can trust."

## Differentiation vs common WP backup plugins
- Outside-in backups via SSH + DB export (less PHP timeout fragility)
- Checksums and manifests by default
- Restore drills as first-class feature
- Storage-provider flexibility (not locked)

## Ideal Early Customers
1. WordPress freelancers with 5-50 sites
2. Small agencies with mixed hosting environments
3. Technical site owners burned by failed restores

## Messaging Angles
- "Backup is easy. Restore is hard. We solve restore."
- "Proof-based backup: checksum + drill-ready."
- "Bring your own cloud. Keep control."

## Channel Strategy
- GitHub + docs + examples
- YouTube demo: backup -> verify -> restore in one take
- WordPress communities + agency forums
- Case studies from real incidents and recoveries
